Teil 2: Kurzantworten (1–2 Sätze)
3. Erkläre mit deinen eigenen Worten, was eine Datenbank ist.
- Eine Datenbank ist eine Sammlung von zusammenhängenden Daten. Sie kann ein einfacher Zettel sein mit Dingen die man einkaufen muss bis hin zu komplexen und riesigen Sammlungen, um alle Daten im Kontext einer E-Commerce Plattform wie Amazon zu speichern.

4. Nenne zwei konkrete Vorteile, die ein DBMS gegenüber einer einfachen Textdatei bietet.
- Ein DBMS übernimmt den Sicherheitsaspekt einer Datenbank
- DBMS können mit anderen Anwendungen interagieren

5. Was ist eine "Database Query" und warum kann sie in komplexen Datenbanken schwieriger werden?
- Eine Database Query ist eine Anfrage an unser DBMS um spezifische Daten von der Datenbank zu erhalten
